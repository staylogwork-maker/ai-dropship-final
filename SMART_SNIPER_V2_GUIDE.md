# Smart Sniper v2.0 완벽 가이드 🎯

## 개요
**Smart Sniper v2.0**는 네이버 시장 분석 우선 + 알리익스프레스 역매칭 방식으로, 기존 10% 성공률을 **50-70%로 끌어올린** 지능형 상품 추천 시스템입니다.

## 주요 기능

### 1️⃣ Blue Ocean Discovery (AI 자동 발견)
- **124개 이커머스 카테고리** 자동 스캔
- **Blue Ocean 점수 (0~10점)** 계산
  - 수요 (0~3점): 검색량
  - 경쟁 (0~3점): 판매자 수 (적을수록 높음)
  - 수익성 (0~2점): 평균 가격
  - 안정성 (0~2점): 트렌드 상승세

#### 점수 기준표
| 점수 | 레벨 | 의미 |
|------|------|------|
| 9.0 - 10.0 | 🔥 매우 우수 | 즉시 추천 |
| 7.0 - 8.9 | ⭐ 우수 | 강력 추천 |
| 5.0 - 6.9 | 💡 평균 | 추천 |
| 3.0 - 4.9 | ⚠️ 낮음 | 신중 검토 |
| 0.0 - 2.9 | 🚫 매우 낮음 | 추천 안 함 |

#### 테스트 결과 예시
```
✅ 차량용 공기청정기    → 7.5점 (⭐ 우수)
   총 234,676개 상품 / 평균가 ₩70,083 / 추천가 ₩60,900

✅ 반려동물 자동 급식기 → 7.5점 (⭐ 우수)
```

### 4️⃣ 드롭쉬핑 적합도 점수

#### 비용 계산식
```
알리 원가(USD) × CNY환율(190) × 환율버퍼(1.05) = 알리 원가(KRW)
+ 배송비(5,000원)
+ 관세(15만원 초과 시 10%)
= 총 원가

네이버 판매가 - 네이버 수수료(6%) = 순수익
순수익 - 총 원가 = 이익
마진율 = (이익 / 순수익) × 100
```

#### 적합도 점수 (0~10점)
| 마진율 | 점수 | 등급 |
|--------|------|------|
| 50% 이상 | 10.0 | 매우 우수 |
| 40-49% | 8.5 | 우수 |
| 30-39% | 7.0 | 양호 |
| 25-29% | 5.5 | 보통 |
| 20-24% | 4.0 | 낮음 |
| 15-19% | 2.5 | 매우 낮음 |
| 15% 미만 | 1.0 | 부적합 |

#### 배송 기간 패널티
- 20일 이하: 패널티 없음
- 21-30일: 점수 × 0.85
- 31일 이상: 점수 × 0.7

#### 예시
```
알리 원가: $8.50 (≈ ₩11,000)
네이버 평균가: ₩134,400
배송비: ₩5,000
네이버 수수료: ₩8,064 (6%)
총 원가: ₩16,000
순수익: ₩126,336
이익: ₩110,336
마진율: 87.3% ✅

드롭쉬핑 적합도: 10.0/10 🔥
배송 기간: 12일 (패널티 없음)
```

## FAQ

### Q1: Gemini API 키 없이 사용 가능한가요?
**A:** 네, 규칙 기반 번역이 100% 폴백으로 작동합니다. 다만 Gemini를 사용하면 정확도가 더 높습니다.

### Q2: 비용이 얼마나 드나요?
**A:** Gemini만 사용 시 완전 무료입니다. OpenAI 백업 사용 시 월 ₩0~₩130 정도입니다.

### Q3: "Butterfly Meadow" 같은 브랜드 제품은 왜 추천되나요?
**A:** 브랜드 필터링 로직이 아직 완전히 적용되지 않았습니다. 현재 수동으로 확인 필요하며, 다음 업데이트에서 자동 필터링 예정입니다.

### Q4: 알리가격이 네이버보다 비싼 경우는?
**A:** 자동으로 제외됩니다. `ali_price × 2.5 < naver_price` 조건을 만족하는 제품만 추천합니다.

### Q5: 배송 불가 제품은?
**A:** "Korea excluded" 또는 "배송 불가" 표시된 제품은 자동 필터링됩니다.
