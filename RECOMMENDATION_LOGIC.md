## 전체 흐름 요약

```
입력: 키워드 (예: "드라이기")
  ↓
[1단계] AliExpress 검색 (최대 50개)
  ↓
[2단계] 안전 필터 (위험 제품 제거)
  ↓
[3단계] 마진 계산 (목표 30% 이상만)
  ↓
[4단계] 이익 순 정렬 + 중복 제거
  ↓
[5단계] 다양성 확보 (가격대별 3개 선택)
  ↓
출력: Top 3 추천 제품
```

## 단계별 상세 설명

### 📍 **1단계: AliExpress 제품 검색**

**목적**: 판매 가능한 제품 후보군 수집

```python
# 무엇을 하나요?
- AliExpress API로 키워드 검색
- 최대 50개 제품 수집

# 어떤 정보를 가져오나요?
{
  'title': '제품명',
  'price': 가격 (USD),
  'url': 'AliExpress 링크',
  'image': '이미지 URL',
  'moq': 최소주문수량
}

# 예시
입력: "hair dryer"
결과: 50개 헤어 드라이어 제품
```

**✅ 통과 기준**: 제품이 1개 이상 검색됨

### 📍 **2단계: 안전 필터 (Safety Filter)**

**목적**: 법적 리스크 있는 제품 사전 제거

```python
# 제거되는 제품 카테고리
❌ 전자제품 (인증 필요)
❌ 의료기기 (식약처 인증)
❌ 식품/건강식품 (수입신고)
❌ 화장품 (화장품법)
❌ 유아용품 (KC 인증)
❌ 안전용품 (KS 인증)

# 예시
입력: 50개 제품
- "Hair Dryer 1500W" → ❌ 제거 (전자제품)
- "Baby Bottle Warmer" → ❌ 제거 (유아용품)
- "Silicone Spatula" → ✅ 통과 (주방용품)
결과: 45개 통과
```

**✅ 통과 기준**: 위험 카테고리에 속하지 않음

### 📍 **3단계: 마진 계산 (Profitability Analysis)**

**목적**: 실제로 돈을 벌 수 있는 제품만 선택

#### 📊 **계산 공식**

```python
# 1. 비용 계산
알리 가격(원) = 가격(USD) × 환율(190) × 1.05 (변동 버퍼)
배송비 = 5,000원 (기본)
관부가세 = 알리가격 × 10% (15만원 이상만)
──────────────────────────────
총 원가 = 알리가격 + 배송비 + 관부가세

# 2. 수익 계산
네이버 평균가 = 한국 시장 조사 결과
네이버 수수료 = 네이버가 × 6%
순수익 = 네이버가 - 수수료
──────────────────────────────
실제 이익 = 순수익 - 총 원가

# 3. 마진율 계산
마진율(%) = (실제이익 ÷ 순수익) × 100
```

#### 💰 **실제 계산 예시**

```
제품: Silicone Spatula Set
─────────────────────────────────────
📥 원가 계산:
  AliExpress 가격:    $5.00
  환율 적용:          $5 × 190 × 1.05 = ₩998
  배송비:             ₩5,000
  관부가세:           ₩0 (15만원 미만)
  ─────────────────────
  총 원가:            ₩5,998

📤 판매가 계산:
  네이버 평균가:      ₩10,000
  네이버 수수료(6%):  -₩600
  ─────────────────────
  순수익:             ₩9,400

💵 이익 계산:
  실제 이익:          ₩9,400 - ₩5,998 = ₩3,402
  마진율:             (₩3,402 ÷ ₩9,400) × 100 = 36.2%

🎯 결과: 36.2% > 30% (목표) → ✅ 통과!
```

#### ⚖️ **마진율 기준**

| 마진율 | 평가 | 설명 |
|--------|------|------|
| **50%+** | 🏆 최우수 | 드롭쉬핑 최적 |
| **40-50%** | ⭐ 우수 | 매우 수익성 높음 |
| **30-40%** | ✅ 양호 | 추천 가능 |
| **25-30%** | ⚠️ 보통 | 주의 필요 |
| **25% 미만** | ❌ 탈락 | 수익성 부족 |

**✅ 통과 기준**: 마진율 ≥ 30%

### 📍 **4단계: 정렬 및 중복 제거**

**목적**: 최고 수익 제품 우선, 같은 제품 제거

#### 4-1. **이익 순 정렬**

```python
# 정렬 기준: 실제 이익 (원화)
제품들.sort(key=실제이익, reverse=True)

# 예시
1. Silicone Spatula 6pcs  → ₩5,200 이익
2. Silicone Spatula 2pcs  → ₩3,402 이익
3. Kitchen Timer           → ₩2,800 이익
```

#### 4-2. **중복 제거**

```python
# 제목 기반 중복 체크
"2Pcs Silicone Spatula Set" 
  vs 
"2pcs   SILICONE   Spatula    Set"
→ 정규화 후 같으면 제거

# 예시
입력:
1. 2Pcs Silicone Spatula - ₩3,402 이익
2. 2Pcs Silicone Spatula - ₩3,300 이익 ← 중복!
3. 6pcs Silicone Spatula - ₩5,200 이익

결과:
1. 2Pcs Silicone Spatula - ₩3,402 이익 (첫 번째만 유지)
2. 6pcs Silicone Spatula - ₩5,200 이익
```

**✅ 통과 기준**: 제목이 다른 제품만 유지

---

### 📍 **5단계: 다양성 확보 (Diversity Selection)**

**목적**: 사용자에게 다양한 선택지 제공

```python
# Top 3 선택 전략

# 1순위: 최고 이익 제품
top_3[0] = profitable_products[0]

# 2순위: 다른 가격대 (20% 이상 차이)
for product in profitable_products[1:]:
    price_diff = |product.price - top_3[0].price| / top_3[0].price
    if price_diff >= 0.2:  # 20% 이상 차이
        top_3[1] = product
        break

# 3순위: 나머지 다른 제품
top_3[2] = 첫 번째와 다른 제품
```

#### 💡 **실제 예시**

```
입력: 15개 수익성 제품

[선택 과정]
1. 최고 이익 → Silicone Spatula (₩5,200, 가격 ₩10,000)
2. 다른 가격대 → Kitchen Timer (₩2,800, 가격 ₩15,000)
   → 가격 차이: (15,000 - 10,000) / 10,000 = 50% ✅
3. 추가 다양성 → Mousepad (₩3,000, 가격 ₩8,500)

최종 Top 3:
┌─────────────────────────────────────────────┐
│ 1. Silicone Spatula  ₩10,000  이익 ₩5,200 │
│ 2. Kitchen Timer     ₩15,000  이익 ₩2,800 │
│ 3. Gaming Mousepad   ₩8,500   이익 ₩3,000 │
└─────────────────────────────────────────────┘
```

**✅ 최종 기준**:
- ✅ 3개 모두 다른 제품
- ✅ 가격대 다양 (20% 이상 차이)
- ✅ 모두 30% 이상 마진

## 평가 기준 요약

### 🎯 **필수 통과 조건**

| 단계 | 조건 | 기준 |
|------|------|------|
| 1. 검색 | 제품 발견 | 1개 이상 |
| 2. 안전 | 법적 리스크 | 위험 카테고리 제외 |
| 3. 마진 | 수익성 | **30% 이상** |
| 4. 중복 | 유니크 | 제목 다름 |
| 5. 다양성 | 가격대 | **20% 이상 차이** |

### 📊 **최종 점수 기준**

```python
제품 점수 = 
  마진율 가중치(50%) +
  이익 금액 가중치(30%) +
  가격 경쟁력 가중치(20%)

# 마진율: 50% → 10점, 30% → 5.5점
# 이익: ₩10,000 → 10점, ₩2,000 → 4점
# 가격: 네이버 평균 대비 저렴할수록 높은 점수
```

## 🎓 핵심 정리

### ✅ **좋은 제품 기준**
- 마진율 30% 이상
- 실제 이익 ₩2,000 이상
- 안전한 카테고리 (주방, 생활, 액세서리)
- 네이버에 경쟁 제품 있음 (시장 검증)

### ❌ **나쁜 제품 기준**
- 마진율 30% 미만
- 전자제품/의료기기/식품 등
- 가격이 너무 높음 (₩50,000+)
- 네이버에 제품 없음 (수요 불확실)

### 🔧 **설정 가능한 값**
```python
config = {

    'shipping_cost_base': 5000,    # 배송비
    'naver_fee_rate': 0.06,        # 네이버 수수료 (6%)
    'customs_tax_rate': 0.10,      # 관부가세 (10%)
}
```
