## 📋 **AliExpress Affiliate 승인 요구사항**

### **필수 정보**
1. **이메일/비밀번호** (AliExpress 계정)
2. **웹사이트 또는 SNS URL**
3. **프로모션 방법 설명** (자유 기술)
4. **월 트래픽 예상** (선택, 추정치 가능)

### **승인 기준 (관대함)**
- ✅ 웹사이트/SNS가 **실제로 존재**하면 OK
- ✅ 드롭쉬핑/E-commerce 설명하면 OK
- ✅ 트래픽 없어도 OK (예상 트래픽만 입력)
- ❌ 허위 정보만 안 되면 됨

### **승인 소요 시간**
- 보통: 1-2일
- 빠른 경우: 즉시 (몇 시간)
- 늦은 경우: 3-4일 (Support Ticket 제출)

## 🚀 **추천 전략**

### **Phase 1: 임시 URL로 빠른 가입** (지금)
```
1. https://portals.aliexpress.com 접속
2. 가입 양식:
   - Website URL: https://5000-ih0it1b01pfxfhxqrd111-0e616f0a.sandbox.novita.ai
   - Category: Dropshipping/E-commerce
   - Promotion Method: "I run an AI-powered dropshipping platform that helps sellers find and import AliExpress products to Korean marketplaces (Naver, Coupang)"
   - Traffic: 100-500 visitors/month (예상치)
3. 제출 → 승인 대기 (1-2일)
```

### **Phase 2: API 키 발급 후 도메인 추가** (승인 후)
```
1. AliExpress Open Platform 로그인
2. App 생성 (API 키 발급)
3. 나중에 도메인 추가:
   - Settings → Website URL 변경
   - 당신의 도메인으로 업데이트
```

### **Phase 3: 도메인 연결 (선택, 나중에)** (여유 있을 때)
```
1. 당신의 도메인 DNS 설정:
   - A Record → 웹앱 서버 IP
   - 또는 CNAME → Sandbox URL
2. AliExpress에서 URL 업데이트
3. 영구 도메인으로 전환 완료
```

## 💡 **중요 팁**

### **1. 임시 URL도 충분함**
- AliExpress는 **웹사이트 존재 여부**만 확인
- Sandbox URL도 **실제 작동하는 웹사이트**이므로 OK
- 나중에 도메인으로 변경 가능

### **2. 가입 시 설명 예시**
```
English (추천):
"I operate an AI-powered dropshipping automation platform that connects 
AliExpress suppliers with Korean e-commerce marketplaces (Naver SmartStore 
and Coupang). Our system automatically imports products, processes images, 
generates marketing content, and manages orders."

Korean (번역 필요 시):
"한국 전자상거래 마켓플레이스(네이버 스마트스토어, 쿠팡)와 AliExpress 공급업체를 
연결하는 AI 기반 드롭쉬핑 자동화 플랫폼을 운영합니다. 우리 시스템은 상품을 자동으로 
가져오고, 이미지를 처리하며, 마케팅 콘텐츠를 생성하고, 주문을 관리합니다."
```

### **3. 트래픽 예상치 (적당히 입력)**
```
- Current Traffic: 100-500 visitors/month
- Expected Traffic (3 months): 1,000-5,000 visitors/month

이유: 아직 초기 단계이므로 낮은 수치 OK
```

### **4. 승인 거부 시 대응**
```
❌ 거부 사유: "웹사이트 트래픽 부족"
✅ 재신청: 도메인 추가 + 트래픽 예상치 높게 수정

❌ 거부 사유: "웹사이트 미완성"
✅ 재신청: Sandbox URL에 더미 콘텐츠 추가
```

## 🎯 **당신의 도메인 활용 방법**

### **당신이 도메인을 **이미** 구매했다면:**

#### **방법 A: 지금 도메인 사용** (10분 추가 작업)
```
1. 도메인 DNS 설정:
   - 도메인 관리 페이지 접속
   - A Record 추가:
     * Name: @ (또는 www)
     * Value: [웹앱 서버 IP]
     * TTL: 3600
2. 가입 시 도메인 입력:
   - Website URL: https://yourdomain.com
3. 완료! (도메인 전파 24시간 소요)
```

```
1. 지금: Sandbox URL로 가입
   - Website URL: https://5000-ih0it1b01pfxfhxqrd111-0e616f0a.sandbox.novita.ai
2. 승인 후: 도메인 연결 작업
```
